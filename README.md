# bs-chai

Bucklescript bindings to chai 

## Installation

Install the project:

`npm install bs-chai --save`

And add the dependency to your bs dependencies in `bsconfig.json`:

```json
"bs-dev-dependencies": [
  "bs-chai"
]
```

It'll be available under the `BsChai` namespace
